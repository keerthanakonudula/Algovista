# 🚀 AlgoVista | Sorting Visualizer

**AlgoVista** is a high-performance, web-based tool designed to visualize the logic behind the world's most famous sorting algorithms. Built with a focus on clean UI/UX and smooth animations.

## 🌟 Features
* **5 Core Algorithms:** Implementation of Bubble, Selection, Insertion, Quick, and Merge Sort.
* **Fluid UI:** A glassmorphism-inspired interface with smooth page transitions.
* **Real-time Control:** Adjust the visualization speed and array size on the fly.
* **Responsive Design:** Fully functional across different screen sizes.

## 🛠️ Tech Stack
* **HTML5** - Structure
* **CSS3** - Styling & Animations (Glassmorphism)
* **JavaScript (ES6+)** - Sorting Logic & DOM Manipulation

## 📊 Algorithms Explained

| Algorithm | Best Case | Average Case | Worst Case | Space Complexity |
| :--- | :--- | :--- | :--- | :--- |
| **Bubble Sort** | $O(n)$ | $O(n^2)$ | $O(n^2)$ | $O(1)$ |
| **Selection Sort** | $O(n^2)$ | $O(n^2)$ | $O(n^2)$ | $O(1)$ |
| **Insertion Sort** | $O(n)$ | $O(n^2)$ | $O(n^2)$ | $O(1)$ |
| **Quick Sort** | $O(n \log n)$ | $O(n \log n)$ | $O(n^2)$ | $O(\log n)$ |
| **Merge Sort** | $O(n \log n)$ | $O(n \log n)$ | $O(n \log n)$ | $O(n)$ |
